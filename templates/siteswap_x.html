Siteswap X: Official Syntax Reference

Siteswap X extends standard siteswap notation to clearly describe advanced throws and catches for competitive and performance juggling.

1. Core Siteswap Syntax

Siteswap numbers represent the number of beats a prop spends in the air. Higher numbers mean higher throws. Letters (a-z) are used for throws above 9. If you are not familiar with siteswap notation, see Thom Wall's article.

2. Throw and Catch Modifiers

Modifiers are written above and below each siteswap digit. The throw modifier (and optional spin) appears above the digit, and the catch modifier appears below the digit. The format is:

{{ sx.siteswap_x_viewer(parse_siteswap_x('3{B/P}'), '1.5em') }}
  • Throw modifier: Above the siteswap digit. A single letter (or letter+spin) describing the throw, e.g. B (Backcross), U2 (Under-the-leg, 2 spins).
  • Catch modifier: Below the siteswap digit. A single letter describing the catch style, e.g. P (Penguin), L (Lazy).
